## 🎨 Дизайн-рекомендации

### Цветовая палитра:

- **Primary:** Черный (#000000)
- **Secondary:** Белый (#FFFFFF)
- **Accent:** Один яркий цвет (например, #6366F1 синий или #EC4899 розовый)
- **Градиенты:** Для фонов и акцентов

### Шрифты:

- **Заголовки:** Inter Bold / SF Pro Display Bold (48-72pt)
- **Основной текст:** Inter Regular / SF Pro Text (18-24pt)
- **Цифры:** Inter/SF Pro Bold (для метрик — крупно!)

### Иконки:

- Используйте минималистичные иконки (Lucide, SF Symbols)
- Единый стиль (outline или filled)

### Изображения:

- Высокое качество screenshots приложения
- Mockups на iPhone/Android устройствах
- Реальные примеры образов из приложения

### Анимации (для цифровой презентации):

- Плавные fade-in для элементов
- Slide-in для графиков
- Hover effects для интерактивных элементов

## 📝 Советы по подаче

### DO:

✅ Говорить уверенно и кратко (30-40 сек на слайд)  
✅ Фокусироваться на цифрах и метриках  
✅ Рассказывать историю, а не просто читать слайды  
✅ Держать зрительный контакт  
✅ Показать демо приложения (если возможно)

### DON'T:

❌ Перегружать слайды текстом  
❌ Использовать сложные графики без пояснений  
❌ Говорить слишком технически (если аудитория не техническая)  
❌ Пропускать слайд с рисками  
❌ Заканчивать без четкого call-to-action

## 🎯 Адаптация под аудиторию

### Для инвесторов:

- Больше фокуса на финансовых метриках (слайды 10-11)
- Акцент на масштабируемость и exit strategy
- Команда и traction

### Для грантов:

- Социальная значимость проекта
- Инновационность технологии (AI)
- Потенциал создания рабочих мест

### Для партнеров:

- Product-market fit
- Growth metrics
- Возможности для коллабораций

### Для медиа:

- Проблема и решение (слайды 2-3)
- Уникальность продукта
- Vision и миссия

### Короткая версия (5 минут, 8-10 слайдов):
