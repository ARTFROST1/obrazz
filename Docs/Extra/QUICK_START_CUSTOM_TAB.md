# Custom Tab - Быстрый старт

## 🚀 Как добавить/удалить категории (за 30 секунд)

### **Простая инструкция:**

```
1. Открыть создание образа
   ↓
2. Нажать таб "⚙️ Custom" (справа)
   ↓
3. Нажать кнопку "Edit" (справа вверху)
   ↓
4. Добавить категорию:
   - Нажать "+ Add Category"
   - Выбрать категорию из списка

   Удалить категорию:
   - Нажать "✕" на категории
   ↓
5. Нажать "Done" (зеленая кнопка)
   ↓
✅ Готово! Карусели обновлены
```

## ⚡ Важные моменты:

✅ **Изменения сохраняются автоматически** - при нажатии Done
✅ **Минимум 1 категория** - нельзя удалить последнюю
✅ **Максимум 8 категорий** - все доступные категории
✅ **Переключение таба** - выходит из Edit режима автоматически

## 🎯 Частые вопросы:

**Q: Где кнопка Edit?**  
A: Справа вверху, когда вы на Custom табе

**Q: Почему не могу удалить категорию?**  
A: Если это последняя - нельзя. Минимум 1 должна остаться

**Q: Как долгое нажатие?**  
A: Пока не реализовано. Используйте Edit → ✕ для удаления
